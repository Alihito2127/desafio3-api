# Desafio 3 - API de Login

## Descrição
API REST de login com 4 funcionalidades:
- Login com sucesso
- Login inválido
- Bloqueio após 3 tentativas
- Lembrar senha

## Como usar

### 1. Instalar dependências
```bash
npm install
```

### 2. Rodar a API
```bash
npm start
```

### 3. Rodar os testes
```bash
npm test
```
